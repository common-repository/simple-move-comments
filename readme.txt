== Description ==

`Simple Move Comments` moves all comments from one page/post to another.
Yust select the source and destination and press the `move comments` button.

All changes will be logged in a logfile.

== Installation ==

1. Upload the `Simple Move Comments` plugin to the `/wp-content/plugins/` directory.
2. Activate the plugin through the *Plugins* menu in WordPress.

== Frequently Asked Questions ==

= Where is the logfile saved? =

The logfile is saved as an ASCII file within the plugin directory.
